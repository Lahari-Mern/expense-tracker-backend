#  Expense Tracker Backend

This is a simple **Expense Tracker Backend API** built using **Node.js, Express, and MongoDB**.  
It performs full CRUD operations (Create, Read, Update, Delete).

---

##  Features

- Add new transaction (Income / Expense)
- Get all transactions
- Update transaction
- Delete transaction
- MongoDB database integration
- REST API using Express

---

##  Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman (for testing)
